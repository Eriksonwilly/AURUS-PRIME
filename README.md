# AURUS PRIME

Aplicación web para **diseño y análisis de muros de contención** (Rankine, Coulomb, diseño de fuste, reportes PDF).

Interfaz inspirada en landing moderna con tema **negro y dorado**.

## Requisitos

- Python 3.9+
- Dependencias en `requirements.txt`

## Instalación local

```bash
pip install -r requirements.txt
streamlit run APP.py
```

O en Windows:

```bash
EJECUTAR_APP.bat
```
